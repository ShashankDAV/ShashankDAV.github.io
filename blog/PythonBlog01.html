Python: Print And Input

Print

1.In Python "print(text)" is used to output the text.

2.It can also be used to print variables but we have to remove the inverted comma(" ") because inverted comma is used for printing strings or text.

3.If we want to print more than 1 string we can give comma(,) or plus sign(+) between them.

The syntax to use it is:

print("Hello World")
print("This is example "+"of multiple text.")
print("This is second example "+"of multiple text.")

Output:

Hello World
This is example of multiple text.
This is second example of multiple text.

Variables

1.In Python variables are like container which are used to store items like integers , strings(text) etc.

2.We can name variable by using any string or text.

3.Note : We can't use integers like 1, 67 , 7 etc and special characters like @ , $ , _ etc to name a vari-able.

The syntax to use it is:

name = "Shashank Pathak"
print(name)

Output:

Shashank Pathak

Input

1.In Python "input(text)" is used to ask for user input.

2.We have to store it in a variable and then we can print this.

The syntax to use it is:

name = input("Enter your name : ")
print("Name you entered : ", name)

Output:

Enter your name : IDK
Name you entered : IDK
